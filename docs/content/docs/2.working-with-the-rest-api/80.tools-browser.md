`Browser`{lang="ts-type"} — это синглтон (`new BrowserManager()`), экспортируемый из `@bitrix24/b24jssdk`. Он читает `navigator.userAgent` один раз во время импорта. В небраузерных окружениях (Node) каждая проверка безопасна — менеджер возвращает `'?'`.

```ts
import { Browser } from '@bitrix24/b24jssdk'

if (Browser.isMobile()) {
  // ...
}
if (Browser.isLocalStorageSupported()) {
  // ...
}
```

## Определение браузера {#browser-detection}

| Метод | Возвращает |
|----|----|
| `isOpera()` | `true`, если UA содержит `opera`. |
| `isIE()` / `isIE6/7/8/9/10/11()` | Эвристики для версий Internet Explorer. |
| `isSafari()` | `safari && !chrome`. |
| `isFirefox()` / `isChrome()` | Очевидно из названия. |
| `detectIEVersion()` | Числовая версия IE (`-1`, если не IE). |

## ОС / платформа {#os-platform}

| Метод | Возвращает |
|----|----|
| `isMac()` / `isWin()` / `isLinux()` | Десктопная ОС. |
| `isAndroid()` | UA содержит `android`. `detectAndroidVersion()` возвращает разобранную версию. |
| `isIPad()` / `isIPhone()` / `isIOS()` | Устройство iOS. `isIPad()` также покрывает Mac с тачем (современный iPadOS). |

## Возможности {#capabilities}

| Метод | Возвращает |
|----|----|
| `isMobile()` | UA iOS / Android / `mobile` / `touch`. |
| `isRetina()` | `window.devicePixelRatio >= 2`. |
| `isTouchDevice()` | `'ontouchstart' in window` или `navigator.maxTouchPoints > 0`. |
| `isDoctype(target?)` | `compatMode === 'CSS1Compat'` (или запасной вариант). |
| `isLocalStorageSupported()` | `setItem/removeItem` round-trip проходит успешно. |
